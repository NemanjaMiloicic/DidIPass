# DidIPass

- Project made for the purposes of monitoring changes on cs.elfak website.
- Automated mini web scraper that monitors changes and commits reports to repository.
- This project runs automatically using GitHub actions on scheduled interval.
- Uses **Python,** **Selenium** and **GitHub actions with cron schedule**.
- Purpose of this project is to reduce the need for frequent manual checking for changes.
